# 🧒 Verificador de Idade

Este é um programa simples em Python que lê duas idades fornecidas pelo usuário e classifica cada uma como:

- 👶 **Criança**: menor de 12 anos  
- 🧑‍🎓 **Adolescente**: entre 12 e 17 anos  
- 🧔 **Adulto**: 18 anos ou mais  

## 🚀 Como funciona

1. O programa pede ao usuário que digite duas idades.
2. Cada idade é armazenada em uma lista.
3. Em seguida, o programa percorre a lista e exibe a classificação correspondente.
